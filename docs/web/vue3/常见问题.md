## 2、vue props 使用 typescript 自定义类型

```ts
import { PropType } from 'vue'

type UserItem = {
  id: string
  userName: string
}

const props = defineProps({
  data: {
    type: Object as PropType<UserItem>,
    default: () => {},
  },
})
```
