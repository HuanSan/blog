# less

Less （Leaner Style Sheets 的缩写） 是一门向后兼容的 CSS 扩展语言。

Less 到底为 CSS 添加了什么功能？以下就是这些新加功能的概览。

## 1、变量

```less
@width: 10px;
@height: @width + 10px;

.header {
  width: @width;
  height: @height;
}

// 编译为
.header {
  width: 10px;
  height: 20px;
}
```

## 2、嵌套

```less
#header {
  color: black;
  .navigation {
    font-size: 12px;
  }
  .logo {
    width: 300px;
  }
}
```

## 3、混合器

```less
// 定义
.bordered {
  border-top: dotted 1px black;
  border-bottom: solid 2px black;
}
// 使用
.menu a {
  color: #111;
  .bordered();
}

.post a {
  color: red;
  .bordered();
}
```

## 4、注释

```less
/* 一个块注释
 * style comment! */
@var: red;

// 这一行被注释掉了！
@var: white;
```

## 5、导入

```less
@import "library"; // library.less
@import "typo.css";
```
