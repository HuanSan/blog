# nodejs使用typescript

nodejs 使用 TypeScript 来编写项目代码

# 1、安装

（1）ts-node 可以直接运行ts文件

npm i ts-node -g

查看版本：

ts-node -v

（2）typescript 编译

npm i typescript -g

查看版本：

tsc -v

# 2、配置 tsconfig.json

在项目根目录创建 tsconfig.json 文件

参考配置：

# 3、配置 package.json

在这里可以配置启动、编译等指令

  "scripts": {
    "start": "node ./dist/app.js", // 运行编译后的代码
    "dev": "ts-node ./src/app.ts", // 运行 typescript 代码
    "build": "tsc", // 编译
    "startup": "tsc && node ./dist/app.js", // 编译并运行
  },
